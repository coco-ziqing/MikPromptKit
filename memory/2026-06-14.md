## 关键发现
1. **依赖注入时机**：`_migrate_v4` 必须在 Seedance V2 数据初始化之后执行，否则 `library_assets` 为空
2. **PyInstaller 模块丢失**：`seed_migrate.py` 未自动打包 → 改为内联到 `main.py`
3. **CREATE TABLE 不幂等**：使用 `CREATE TABLE IF NOT EXISTS` 后，旧表中缺失的列需通过 `ALTER TABLE` 补丁
4. **前端模块名翻译**：4 处直接显示 `card.module` 原始 ID，需加 `_moduleDisplayName()` 翻译
