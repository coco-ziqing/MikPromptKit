## 二、竞品调研总结 (2026-06-02)

### 对标工具分类

| 类别 | 工具 | 核心能力 |
|------|------|---------|
| 企业Prompt管理 | Braintrust / PromptLayer / LangSmith | 版本管理、环境部署、A/B测试、Eval体系 |
| 开源LLMOps | Agenta(MIT) / Langfuse(MIT) / Arize Phoenix | Git式版本管理、可观测性、评估流水线 |
| 轻量Prompt工具 | AI Gist / PromptPal / Instruere | 本地优先、模板变量、Docker部署 |
| 单词卡/SRS | Anki / RemNote / Brainscape / Quizlet | 间隔重复算法、学习模式多样化、卡片模板 |
| 媒体资产DAM | Adobe AEM / Brandfolder / Veritone | 元数据框架、AI自动标签、受控词表 |

### PromptKit 差异化优势（护城河）

1. 🔥 **媒体+提示词一体化** — 竞品均不具备图片/视频管理能力
2. 🔥 **完全离线 + 局域网部署** — Windows原生一键启动，企业内网刚需
3. 🔥 **语义搜索 + Ollama本地翻译** — 无需联网
4. 🔥 **PNG含元数据导出 / .pt包系统** — 业界独创
5. 🔥 **110+ REST API** — 可被外部系统集成

| 缺失功能 | 行业认可度 | 预估工时 |

## 四、核心战略判断（2026-06-02 深度分析）

### 最大问题

> **PromptKit 当前的问题是"存了很多，用得很少"。**

177 条词条，使用计数仅 12 次——用户只是导入、浏览，但没有真正"使用"提示词。工具的前半段（创作→存储→管理）覆盖良好，但后半段（使用→分析→优化）断层严重。

### 最高杠杆升级方向

```
          当前覆盖 45%
创作 → 存储 → 管理 → 使用 → 分析 → 优化
                      ↑ 这里断层 ↑
```

**三个 2 天内可完成的改进，能从根本上改变用户行为：**

#### 🎯 #1 复制时加评分反馈（0.5天）
- 复制后弹窗："这个提示词效果如何？👍 👎"
- 扩展 `usage_history` 表增加评分字段
- 积累真实使用数据 → 统计可展示

#### 🎯 #2 AI优化建议（0.5天，利用已有Ollama）
- 编辑弹窗增加"AI建议"按钮
- Ollama分析提示词清晰度、完整性、结构
- 返回优化建议 + 一键应用

#### 🎯 #3 质量评分体系（1天）
- `quality_score = 清晰度×0.3 + 使用频率×0.2 + 用户评分×0.3 + 独特性×0.1 + 媒体×0.1`
- 卡片显示质量徽章 🔥 85/100
- 低分词条提示"建议优化或删除"

### 其他高价值方向（按优先级）

#### 🟠 中期方向
- Playground 增强（测试历史 + 对比）
- 提示词变体管理（同词多条版本）
- 媒体证据链（图片→关联使用记录）

#### 🟡 远期方向
- 知识图谱化（基于标签关联）
- 今日推荐引擎（SRS + 使用模式）
- 全键盘操作 / 导出格式扩展

## 六、产品定位

**"AI创作者的本地媒体+提示词一体化工作站"**

核心定义：个人/小团队的AI提示词创作、管理、测试、媒体关联的一站式本地工具。
- 不是企业协作平台（不做Braintrust/Agenta的竞争对手）
- 不是纯学习工具（不做Anki替代品）
- 而是**本地离线 + 媒体集成 + Windows原生**的差异化赛道
