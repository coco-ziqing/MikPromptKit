## 一、当前系统能力评分

| 维度 | 评分 | 说明 |
|------|------|------|
| **提示词CRUD** | ⭐⭐⭐⭐⭐ | 82个API端点，覆盖全面 |
| **分类/标签** | ⭐⭐⭐⭐ | 模块+分类+子分类+标签，但无自动建议 |
| **搜索** | ⭐⭐⭐⭐ | FTS5全文搜索+分类筛选，但无语义搜索 |
| **导出导入** | ⭐⭐⭐⭐⭐ | PNG含元数据/.pt包/JSON/TXT，业界独创 |
| **媒体管理** | ⭐⭐⭐⭐ | 缩略图/视频上传裁剪压缩查看器 |
| **批量操作** | ⭐⭐⭐⭐ | 复制/导出/删除/加词包 |
| **数据备份** | ⭐⭐⭐⭐ | 自动每小时+7天轮换 |
| **UI/UX** | ⭐⭐⭐ | 深色模式+列数调节+F5保持，但移动端弱 |
| **版本管理** | ⭐ | **无** — 提示词修改无历史记录 |
| **LLM集成** | ⭐ | **无** — 不能直接发送给AI测试 |
| **协作** | ⭐ | **无** — 单用户 |
| **可观测性** | ⭐ | **无** — 无token统计/成功率 |
| **模板变量** | ⭐ | **弱** — Seedance组装器有限支持 |

## 二、对标开源项目对比

### 2.1 Langfuse（最成熟的LLMOps平台）
- ✅ 提示词版本管理（Git风格diff）
- ✅ LLM Playground（在线测试提示词）
- ✅ 可观测性（token使用/延迟/成本追踪）
- ✅ 多环境部署（dev/staging/prod）
- ❌ **无本地PNG导出**
- ❌ **无媒体缩略图管理**
- ❌ 重度依赖PostgreSQL+Node.js
- ⚠️ **PromptKit差异化优势**：轻量（SQLite）、PNG包系统、纯局域网

### 2.2 Agenta
- ✅ 可视化Prompt编辑器
- ✅ 批量评估（eval pipeline）
- ✅ 版本对比
- ❌ 云服务依赖，自部署复杂
- ❌ 无PNG/离线导出

### 2.3 PromptHive（CLI-first）
- ✅ 终端操作，毫秒级响应
- ✅ Pipe到任意AI工具
- ✅ Git集成
- ❌ 无WebUI
- ❌ 无媒体管理

### 2.4 Promptcat
- ✅ 单HTML零依赖
- ✅ 浏览器本地存储
- ❌ 功能极简，无后端

## 三、差距分析 & 升级建议

### 🔴 P0 — 核心功能缺失（必须补）

| 功能 | 建议方案 | 预估工时 |
|------|---------|---------|
| **提示词版本管理** | 新增`prompt_versions`表，每次编辑自动存档，前端新增"历史版本"弹窗 | 2天 |
| **版本对比（diff）** | 前端引入diff库（diff-match-patch），可视化显示版本差异 | 1天 |
| **语义搜索** | 集成 sentence-transformers 或本地向量数据库（chroma/sqlite-vss） | 2天 |
| **LLM Playground** | 后端新增 `/api/playground/test` 端点，前端弹窗式测试面板 | 1.5天 |

### 🟡 P1 — 体验增强（推荐补）

| 功能 | 建议方案 | 预估工时 |
|------|---------|---------|
| **移动端适配** | CSS 媒体查询优化，折叠侧栏，触摸友好按钮 | 1天 |
| **标签自动补全** | 输入标签时从已有标签库建议 | 0.5天 |
| **批量拖拽排序** | 收藏夹/词包内卡片可拖拽排序 | 1天 |
| **Markdown导出** | 批量导出增加Markdown格式 | 0.5天 |
| **提示词模板变量** | `{{variable}}` 语法 + 运行时替换 | 1天 |
| **快捷键系统** | `Ctrl+E`编辑 `Ctrl+D`删除 `Ctrl+N`新建 | 0.5天 |
| **批量标签编辑** | 勾选多个词条后批量添加/移除标签 | 0.5天 |
| **统计仪表盘** | 各模块使用频率TOP、趋势图 | 1天 |
| **导出格式扩展** | CSV / Markdown / OpenAI格式 | 0.5天 |

### 🟢 P2 — 锦上添花（有时间再做）

| 功能 | 建议方案 | 预估工时 |
|------|---------|---------|
| **Prompt Run 记录** | 每次复制/测试记录时间、结果、token数 | 1天 |
| **多用户支持** | 简易用户管理（local auth），基于用户的收藏/词包隔离 | 3天 |
| **WebHook/API** | 新增REST API key，供外部程序调用 | 1天 |
| **浏览器扩展** | Chrome/Firefox插件从任意页面右键保存提示词 | 3天 |
| **i18n 多语言** | 前端国际化支持（en/ja/ko） | 2天 |
| **TG/微信机器人** | 通过Bot接收/检索/发送提示词 | 2天 |
| **批量OCR导入** | 从截图/图片识别提示词文本 | 1天 |
| **PWA离线支持** | Service Worker + Cache API，无网络可用 | 1天 |

## 六、PromptKit 差异化护城河（不要放弃的优势）

| 优势 | 同类罕见程度 |
|------|-------------|
| **PNG 含元数据导出/导入** | ⭐⭐⭐⭐⭐ — 唯一支持 |

| **纯局域网零依赖** | ⭐⭐⭐⭐ — 安全/离线 |
| **视频+缩略图管理** | ⭐⭐⭐⭐ — 独特 |
| **SQLite单文件** | ⭐⭐⭐⭐ — 极简部署 |
| **Seedance视频模板** | ⭐⭐⭐ — 垂直场景 |
| **批量导出命名+目录选择** | ⭐⭐⭐ — 用户体验细节 |

**升级策略**：在保持以上差异优势的基础上，补齐版本管理+搜索+Playground的短板。
