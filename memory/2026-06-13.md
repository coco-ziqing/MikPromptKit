## 4. 侧边栏竞态刷新修复
- **根因**：`switchSeedanceTab('composer')` → `init()` 和 `openInComposer` → `await init()` 双重调用形成竞态，旧数据 renderProjectList 覆盖新数据
- **修复**：
  - `init()` 加 `_initPromise` 防重入锁
  - `switchSeedanceTab` 增加 `_skipSwitchInit` 标记检查
  - `openInComposer` 设标记 → 跳过 switch 内的 init → 由自己单次 await init + loadProjects + renderProjectList

## 5. 智能导入引擎
- `backend/api/seedance_v2.py`：`POST /import-from-template`

  - 提取全局风格/画幅/分辨率/负面提示词
  - 27 维词卡双向子串匹配（中文分词 > 词卡中文字段截取 > 2-gram 60%阈值）
  - 未匹配描述词 → 自动创建自定义分组+词卡
- `frontend/static/js/app.js`：`openInComposer` 调用 `/import-from-template` 替代原始手动解析
