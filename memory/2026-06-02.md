## Scroll position fix after delete/save/edit — Implementation

**Problem**: After clicking delete card button (`trashPrompt`), `loadPrompts()` → `renderPrompts()` → `container.innerHTML = html` resets scroll to top. Same for `deleteEditPrompt()`, `saveEditPrompt()`, and `generateComfyThumbnail()`.

**Solution**: Modify `loadPrompts()` to save/restore scroll position. This is a single touch point.

In `loadPrompts()`: before fetch, save `window.scrollY`. After `renderPrompts()`, restore via `requestAnimationFrame(() => window.scrollTo(0, savedY))`.

**Fix locations in app.js**:
- `deleteEditPrompt()` (~line 3422): Add scroll save before `await this.loadPrompts()`
- `trashPrompt()` (inline button on card, function needs to be found): Same fix
- `saveEditPrompt()` (~line 3370): Same
- `generateComfyThumbnail()` (~line 3054): Same

**Implementation**: The cleanest approach is to inject scroll save/restore into `loadPrompts()` itself since all these functions call it.

```javascript
async loadPrompts() {
    var savedScroll = window.scrollY;
    // ... existing fetch and render code ...
    // After container.innerHTML = html and other rendering:
    if (savedScroll > 0 && this.state.currentView === 'home') {
        requestAnimationFrame(function() {
            window.scrollTo(0, savedScroll);
        });
    }
}
```

### Code changes in loadPrompts():

## 2026-06-02 19:28 — PNG 拖拽导入修复（BUG: 非编辑模式下拖入 PNG 卡死）
### 现象
- 非编辑模式下拖入 PNG 图片词卡 → 页面卡住 / 跳转到空白页
- 编辑模式下拖拽才起效（但实际上也不起作用，只是没触发浏览器默认行为）

### 根因
1. `init()` 中调用 `this._initDropZone()`（line 98）— **函数未定义**（仅声明了 `_dropAttached: false`）
2. `renderPrompts()` 末尾调用 `this.bindCardDragDrop()`（line ~3658）— **函数未定义**
3. 因为没有 `_initDropZone()` 给 `document` 添加 `dragover`/`drop` 的 `preventDefault()` → 浏览器接管拖入操作 → 尝试打开 PNG 文件为新页面 → 页面卡死

### 修复方案
在 `_onDropPng` 和 `_escape` 之间插入两个函数：

```javascript
    // --- 全局拖拽导入 init ---
    _initDropZone: function() {
        // 全局阻止浏览器默认拖拽行为
        document.addEventListener('dragover', function(e) { e.preventDefault(); });
        document.addEventListener('drop', function(e) { e.preventDefault(); });

        // 创建覆盖层 drop zone
        var zone = document.getElementById('globalDropZone');
        if (!zone) {
            zone = document.createElement('div');
            zone.id = 'globalDropZone';
            zone.style.cssText = 'display:none;position:fixed;top:0;left:0;width:100%;height:100%;z-index:99998;pointer-events:none;';
            document.body.appendChild(zone);
        }

        var count = 0;
        document.addEventListener('dragenter', function(e) {
            count++;
            if (count === 1) {
                zone.style.display = 'flex';
                zone.style.background = 'rgba(99,102,241,0.06)';
                zone.style.border = '3px dashed #6366f1';
                zone.style.alignItems = 'center';
                zone.style.justifyContent = 'center';
                zone.style.pointerEvents = 'auto';
                zone.innerHTML = '<div style="background:rgba(255,255,255,0.95);padding:24px 36px;border-radius:16px;box-shadow:0 8px 32px rgba(0,0,0,0.15);text-align:center;">'
                    + '<div style="font-size:48px;margin-bottom:8px;">📥</div>'
                    + '<div style="font-size:18px;font-weight:600;color:#1e293b;">松开放入 PNG 提示词卡片</div>'
                    + '<div style="font-size:13px;color:#64748b;margin-top:4px;">支持 .png 文件（含提示词元数据）</div>'
                    + '</div>';
            }
        });

        document.addEventListener('dragleave', function(e) {
            count--;
            if (count <= 0) {
                count = 0;
                zone.style.display = 'none';
                zone.style.background = '';
                zone.style.border = '';
                zone.innerHTML = '';
            }
        });

        zone.addEventListener('drop', function(e) {
            e.preventDefault();
            e.stopPropagation();
            count = 0;
            zone.style.display = 'none';
            zone.style.background = '';
            zone.style.border = '';
            zone.innerHTML = '';
            App._onDropPng(e);
        });

        this._dropAttached = true;
    },

    // --- 卡片拖拽排序绑定 ---
    bindCardDragDrop: function() {
        var cards = document.querySelectorAll('#promptList .prompt-card');
        for (var i = 0; i < cards.length; i++) {
            cards[i].setAttribute('draggable', 'false');
        }
    },
```

插入位置：`_onDropPng` 函数（结束于 ~line 5493）之后，`_escape` 函数之前。

### 作用
- `_initDropZone()`: 全局 `preventDefault()` 阻止浏览器默认打开/下载文件；显示覆盖层"松开放入"UI；调用 `_onDropPng()` 导入
- `bindCardDragDrop()`: 编辑模式下防止浏览器原生拖拽（`draggable=false`）
