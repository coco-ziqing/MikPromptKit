# Seedance V2 提示词组装器升级分析 (2026-06-06)

## 当前架构状态

### 后端 (backend/api/seedance_v2.py, ~1034行)
- 27 套维度词库 (prompt_library) — 5 basic + 14 extended + 8 global
- 词卡管理 (prompt_word_card) — CRUD + 搜索 + 热权重排序
- 分镜项目管理 (user_project / user_project_scene)
- 镜头-词卡关联 (user_scene_prompt)
- 时间重算引擎 _recalculate_scene_times() — 锁定/解锁/均分逻辑
- 提示词拼接引擎 compose — 按公式: 运镜+主体+动作+场景+构图+光影+焦段+质感+速率+氛围+特效+细节
- 反向解析 parse — 纯文本→结构化镜头
- 自定义词条 — 用户新增词卡持久化
- 智能推荐 recommend — 空字段推荐补齐词条

### 前端 (frontend/static/js/seedance_v2_composer.js, ~200行 / 39KB)
- 三栏布局：左栏项目列表 + 中栏编辑器 + 右栏（未实现常驻词库面板）
- 项目管理 CRUD + 批量删除 + 弹窗确认
- 分镜卡片渲染 + 时长编辑 + 锁定/解锁 + 拖拽排序 + 复制/插入/删除
- 弹窗式词卡选择器 (s2CardPicker) + 词库Tab切换 + 搜索过滤 + 手动输入
- 时长分配冲突处理 — 4种选择弹窗（新建镜头/修改总时长/解锁其他/直接锁定）
- 输出面板 — 刷新/复制提示词/复制JSON/填入LibTV

### 数据库 (database.py)
- prompt_library — 词库定义表
- prompt_word_card — 词卡表 (is_system + heat_weight + usage_count)
- user_custom_word — 用户自定义词条
- user_project — 分镜项目
- user_project_scene — 镜头数据 (22个字段)
- user_scene_prompt — 镜头-词卡多对多关联
- sys_global_config — 系统配置

### 种子数据 (seedance_v2_seed.py, ~21KB)
- 27套词库定义 + 初始词卡
- 已接入数据库初始化流程

## V1 旧版（仍存在）
- backend/api/seedance.py — 简单 compose 接口
- 前端 old composer 表单 (index.html seedanceComposer)
- 与V2并存，实际由V2接管

## 6 大升级方向

| # | 方向 | 说明 | 难度 |
|---|------|------|------|
| 1 | **右侧持久词库面板** | 弹窗选词 → 编辑区右侧常驻词库面板，拖拽词卡到镜头字段 | 中 |
| 2 | **交互式时间轴条** | 纯展示 → 可拖拽分段的时间轴，支持拆分/合并/微调 | 高 |
| 3 | **分镜模板预设** | 一键套用产品特写/人物出场/风景大景等常见镜头配置 | 中 |
| 4 | **多格式输出** | 支持 Seedance/Kling/Sora/Runway/Pika 五种平台格式 | 中 |
| 5 | **智能填充** | 基于场景描述自动推荐空字段补全 | 低 |
| 6 | **分镜级音频维度** | 每镜头独立音效/配乐字段 | 低 |

## 开发优先级建议
1. 右侧持久词库面板（体验提升最大）
2. 多格式输出（实用价值最高）
3. 分镜模板预设（降低使用门槛）
4. 交互式时间轴条（体验深度）
5. 智能填充 + 分镜音频（锦上添花）

## 已实施优化

### v3.10.25 — 词库弹窗交互优化 (2026-06-06)
- **selectCard**: 选词后不再关闭弹窗，可连续选择不同维度词卡
- **switchPickerLib**: 切换词库Tab时自动更新 activeField，选词写入对应字段
- **closePicker**: 新建统一关闭方法，关闭时自动刷新镜头数据+重算输出
- **关闭触发**: X按钮 / 点击窗口外 / ESC键 三种方式
- **标题优化**: 显示 "✏️ 镜头N - 维度名称"，当前编辑目标一目了然
- **customInput**: 同步优化，手动输入后也不关闭弹窗
- **已选词卡绿色高亮**: 当前维度已选词卡 → 绿色边框+浅绿底+ ✓已选 标记
- **词库Tab绿点标记**: 维度已填充的Tab旁显示绿色圆点
- **key映射修复**: scene→scene_desc、env_detail→environment_detail 双向映射，修复场景词/人文环境词无法写入镜头字段的bug
- **拓展区交互重构**: 拓展维度改为「下拉选分类+➕添加按钮」模式，点击小三角展开/收起控制面板
- **自定义分组系统**: 可在拓展区创建/删除自定义分组词库，词库弹窗底部自动显示添加输入框
- **词库弹窗增强**: 自定义分词组显示在词库弹窗底部 Tab 区，含「⚙管理」按钮；弹窗内可对自定义分组直接添加词条(词条+释义)
