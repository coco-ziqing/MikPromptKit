## 第三轮：缩略图系统重写（11:00~11:28）

### 根因链（三层定位）
```
用户点击「从图库选」→ word_editor._openThumbLibrary()
  → App._onThumbnailSelected = callback    ← 设置回调
  → App._openThumbnailModal('images')       ← 立即清空回调！
     → this._onThumbnailSelected = null;    ← ❌ 致命清空
     → this._onVideoSelected = null;

图库点击图片 → App.selectThumbnail(filename)
  → if (this._onThumbnailSelected) ...     ← null，跳过
  → fallback promptId / collectionId       ← 都是 null
  → 静默失败 ❌
```

### 修复
| 文件 | 改动 |
|------|------|
| `app_media.js` | `_openThumbnailModal` 不再清空回调（调用者提前清理） |
| `word_editor.js` | `_openThumbLibrary` 先设回调再打开modal |
| `word_editor.js` | 上传/图库/清除后同步 `await App.loadPrompts()` 刷新卡片 |
| `app_editor.js` | word_card 有缩略图时编辑模式正常显示 ✕/🔍 按钮 |
