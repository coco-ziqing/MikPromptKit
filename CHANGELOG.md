# Changelog — PromptKit

## v3.1.3 (2026-05-31) — 提示词模板变量

### 📝 模板变量系统
- **`{{variable}}` 语法**：提示词中嵌入 `{{style}}`、`{{subject}}` 等运行时变量
- **自动识别填充弹窗**：点击复制时自动检测模板变量，弹出填充分窗
- **实时预览**：弹窗顶部显示变量替换后的效果预览
- **未填充提醒**：复制时如果有变量未填充，自动警告提示
- **后端 API**：`POST /api/v2/templates/parse` + `POST /api/v2/templates/render`

## v3.1.2 (2026-05-31) — LLM Playground + 体验打磨

### 🎮 LLM Playground
- **Ollama 支持**：本地模型在线测试（`http://localhost:11434`）
- **OpenAI 兼容 API**：支持任何 OpenAI 格式的远程 API
- **配置持久化**：provider / model / temperature / system_prompt 存储到数据库
- **Token 统计**：输入/输出 token 数显示
- **深色终端风格结果区**：LLM 响应以等宽字体展示
- **后端 API**：`GET/POST /api/playground/config` + `POST /api/playground/test`

### ⌨️ 快捷键系统
- `Ctrl+N` 新建提示词 | `Ctrl+E` 切换编辑模式
- `Ctrl+D` 批量删除 | `Ctrl+S` 保存编辑弹窗
- `?` 显示快捷键帮助面板

### 🏷️ 标签自动补全 + 批量标签编辑
- 编辑弹窗标签输入框自动建议已有标签（269 个）
- 编辑模式工具栏「批量标签」按钮：批量添加/移除标签
- 后端 API：`GET /api/v2/tags/list` + `POST /api/v2/tags/batch`

### 📊 统计仪表盘
- 概览卡片：总词条 / 今日使用 / 收藏 / 回收站
- 模块分布：横向柱状图展示各模块词条数
- 使用频率 TOP 10：按使用次数排名
- 标签云 TOP 20：字号随使用量变化
- 后端 API：`GET /api/v2/stats/dashboard`

### 📱 移动端适配
- 768px 以下汉堡菜单 `☰` + 侧栏遮罩层
- 导航栏收缩、列数滑块隐藏、卡片单列
- 批量操作栏水平滚动、弹窗 95vw 自适应
- 筛选栏自动堆叠、自动关闭侧栏

### 🧠 v3.1.1 — 语义搜索
- **Sentence-BERT 向量搜索引擎**：基于 `all-MiniLM-L6-v2` 模型，384维语义向量
- **搜索框一键切换**：🔤 关键词 ↔ 🧠 语义搜索，点击即可切换
- **搜索结果带相似度**：以百分比显示语义匹配程度
- **自动索引**：服务启动时后台重建全量向量索引（170条约15秒）
- **增量更新**：新建/编辑提示词时实时同步向量
- **后端 API**：`POST /api/v2/search/semantic` + `GET /api/v2/search/status` + `POST /api/v2/search/rebuild`

### 📋 v3.1.0 — 提示词版本管理
- **自动版本存档**：每次编辑提示词自动保存历史版本到 `prompt_versions` 表
- **历史版本弹窗**：编辑弹窗底部「📋 历史版本」按钮，列表展示所有版本
- **版本恢复**：一键恢复到任意历史版本，恢复前自动存档当前版本
- **版本对比（diff）**：直观显示两个版本间的字段级差异（旧/新对照）
- **后端 API**：`GET /api/v2/versions/{id}` / `POST .../restore/{vid}` / `GET .../diff/{v1}/{v2}`

### 🎉 新增功能

#### .pt 提示词包系统（PromptKit Package）
- **导出**：支持单条/批量导出为 `.pt` 格式（ZIP 压缩包），内含：
  - `metadata.json` — 提示词数据（module/category/content/meaning/scene/tags）
  - `thumbnails/` — 缩略图文件（自动打包）
  - `videos/` — 视频文件（自动打包）
- **导入**：
  - 拖拽 `.pt` 文件到列表区自动预览
  - 导入面板支持 `.pt` 类型
  - 预览时显示模块下拉选择，支持编辑后导入
  - 重复策略：跳过 / 重命名新增（默认） / 覆盖
- **文件名**：`内容前12字_提示词包_时间.pt`

#### 导出文件名优化
- JSON/TXT 导出：文件名自动取**第一条内容前12字符**（过滤非法字符）
- 多条时追加 `_等N条`
- 覆盖场景：批量模式、编辑模式、预览导出、导入/导出面板

#### 拖拽导入增强
- 支持 `.json` + `.pt` 两种格式拖拽
- 模块字段改为**下拉选择框**，自动读取已有功能模块
- 模块默认取 JSON 记录的原始值，可手动修改
- 深色/浅色主题完整适配

#### 编辑模式增强
- 工具栏新增：批量复制、预览导出、导出 TXT/JSON、导出 .pt
- 侧边栏新增「📚 全部词库」入口
- 新增 `switchAllModules()` 重置所有筛选
- 新建词条自动归属当前浏览的模块

#### 编辑模式筛选栏
- 关键词快速筛选（回车确认）
- 模块筛选下拉
- 收藏状态筛选（所有/已收藏/未收藏）
- 匹配计数 + 重置按钮

### 🔧 优化
- 全站内联硬编码颜色移除，改用 CSS 变量
- 新增 `--hover-bg` 主题变量
- 导出预览弹窗样式优化
- 拖拽遮罩层视觉优化（模糊背景 + 虚线边框）

### 🐛 Bug 修复
