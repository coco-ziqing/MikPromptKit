# Changelog — PromptKit

## v3.0.0.2 (2026-05-31) — .pt 包系统 + 导出名称优化 + 拖拽增强

### 🎉 新增功能

#### .pt 提示词包系统（PromptKit Package）
- **导出**：支持单条/批量导出为 `.pt` 格式（ZIP 压缩包），内含：
  - `metadata.json` — 提示词数据（module/category/content/meaning/scene/tags）
  - `thumbnails/` — 缩略图文件（自动打包）
  - `videos/` — 视频文件（自动打包）
- **导入**：
  - 拖拽 `.pt` 文件到列表区自动预览
  - 导入面板支持 `.pt` 类型
  - 预览时显示模块下拉选择，支持编辑后导入
  - 重复策略：跳过 / 重命名新增（默认） / 覆盖
- **文件名**：`内容前12字_提示词包_时间.pt`

#### 导出文件名优化
- JSON/TXT 导出：文件名自动取**第一条内容前12字符**（过滤非法字符）
- 多条时追加 `_等N条`
- 覆盖场景：批量模式、编辑模式、预览导出、导入/导出面板

#### 拖拽导入增强
- 支持 `.json` + `.pt` 两种格式拖拽
- 模块字段改为**下拉选择框**，自动读取已有功能模块
- 模块默认取 JSON 记录的原始值，可手动修改
- 深色/浅色主题完整适配

#### 编辑模式增强
- 工具栏新增：批量复制、预览导出、导出 TXT/JSON、导出 .pt
- 侧边栏新增「📚 全部词库」入口
- 新增 `switchAllModules()` 重置所有筛选
- 新建词条自动归属当前浏览的模块

#### 编辑模式筛选栏
- 关键词快速筛选（回车确认）
- 模块筛选下拉
- 收藏状态筛选（所有/已收藏/未收藏）
- 匹配计数 + 重置按钮

### 🔧 优化
- 全站内联硬编码颜色移除，改用 CSS 变量
- 新增 `--hover-bg` 主题变量
- 导出预览弹窗样式优化
- 拖拽遮罩层视觉优化（模糊背景 + 虚线边框）

### 🐛 Bug 修复
- 新建词条不归属当前模块 ✓
- 拖拽导入后词条不可见（模块筛选冲突） ✓
- `.pt` 预览 content 被截断 80 字 ✓
- `.pt` 导入预览编辑不生效 ✓
- 深色主题下导入弹窗文字不可见 ✓
- 导出文件名中文编码问题（Content-Disposition） ✓

### 📊 项目指标

- 代码总量：**~8954 行**（Python 3446 + JS 3768 + CSS 1239 + HTML 501）
